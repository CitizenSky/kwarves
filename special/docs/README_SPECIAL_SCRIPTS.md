Spezialskripte im Ordner scripts/special
========================================

Dieser Ordner enthaelt Spezialwerkzeuge, die nicht zur allgemeinen
regelmaessigen Hauptpipeline gehoeren.

Die Dokumentation liegt getrennt von den Skripten in:

  /Users/koni/astro_projects/scripts/special/docs

Aktuell:

  - compute_hz.py
    Manuelles Spezialwerkzeug fuer Habitable-Zone-Berechnungen.

  - spc_art_vetter.py
    Einzelkandidaten-Vetter fuer starke oder strittige Kandidaten.

  - tic13096842_analysis.py
    Zentraler Spezialstarter fuer TIC 13096842.

  - tic13096842_sector98.py
    Zentraler Spezialstarter fuer die Sector-98-Analysen zu TIC 13096842.
