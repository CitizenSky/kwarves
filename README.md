# kwarves

Independent citizen-science exoplanet project.

## Live site

GitHub Pages (main branch, root folder):

- https://citizensky.github.io/kwarves/

## Deployment

This repository is deployed with **GitHub Pages only**.

Current publish flow:

1. Push to `main`
2. GitHub Pages serves from `/(root)`
3. Root `index.html` redirects to `dashboard/index.html`

No Netlify or Cloudflare deployment is used.
